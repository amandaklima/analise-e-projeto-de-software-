#  Alfabetiza – Aprendendo a aprender

> Aplicação Web desenvolvida para a disciplina de Análise e Projeto de Software – Universidade Católica de Pernambuco  

---

##  Visão Geral

O **Alfabetiza** é uma aplicação web interativa voltada para o **combate ao analfabetismo funcional**. Oferecendo uma abordagem interativa e adaptada às necessidades individuais de cada usuário. Permitindo que os usuários conquistem habilidades de leitura e escrita de forma eficaz.
---

##  Alinhamento com os ODS da ONU

Este projeto contribui diretamente com o **Objetivo de Desenvolvimento Sustentável 4: Educação de Qualidade**, especialmente com a **Meta 4.6**:

> "Até 2030, garantir que todos os jovens e uma substancial proporção dos adultos, homens e mulheres, atinjam a alfabetização e o domínio da matemática."

🔗 [Conheça o ODS 4](https://brasil.un.org/pt-br/sdgs/4)
